💻 Laptop Price Prediction

📖 Project Overview

This project aims to predict the price of a laptop based on its specifications using Machine Learning. The dataset contains information such as the laptop company, processor, RAM, storage, display, GPU, operating system, weight, and price.

The project follows a complete machine learning workflow, including data preprocessing, feature engineering, exploratory data analysis (EDA), model training, evaluation, and deployment.

---

📊 Dataset Features

- Company
- TypeName
- Inches
- Screen Resolution
- CPU
- RAM
- Memory
- GPU
- Operating System
- Weight
- Price (Target Variable)

🔧 Feature Engineering

The following preprocessing steps were performed:

- Extracted CPU Brand
- Extracted GPU Brand
- Converted RAM and Weight into numerical values
- Extracted SSD, HDD, Flash Storage, and Hybrid Storage capacities
- Extracted Screen Resolution (X Resolution and Y Resolution)
- Created PPI (Pixels Per Inch) feature
- Extracted Touchscreen and IPS Display information
- Encoded categorical variables
